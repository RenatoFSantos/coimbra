
# Template de Prompt para Criar uma landpage do projeto.

Crie o conteúdo da landpage dentro da rota principal substituindo o arquivo "@app/page.tsx"

## Descrição

- Vamos utilizar a página criada no Figma como fonte: https://www.figma.com/design/ttrXEB6SwaB3nuJohPp3zD/Coimbra?node-id=1-2&m=dev
- Seja fiel a todas as informações, fontes, layouts e demais conteúdos da página
- Não precisa colocar ações nos botões ou opções de menus
